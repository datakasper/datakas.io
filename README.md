# Datakas website

Static marketing website for Datakas.

## Local preview

Open `index.html` directly in a browser, or serve the folder with a simple local web server.

## Deployment

This site is designed for GitHub Pages with the custom domain `datakas.io`.

Files to deploy:

- `index.html`
- `styles.css`
- `script.js`
- `CNAME`
- `assets/`
